Clone the repository:
1. git clone https://github.com/maaswin03/Final.git
2. cd FInal

Setting Up the Frontend (React):
1. cd frontend
2. npm install
3. npm run dev 

Setting Up the Backend (Node.js):
1. cd Backend_Node
2. npm install
3. node index.js

Setting Up the Backend (Flask):
1. Cd Backend_Python
2. pip install -r requirements.txt
3. python app.py
